# AutoTags v0.0.32（2025-10-28）

## 变更概览

- 支持直接创建标签模式

## 详情

### 新增
- 新增独立模式，使用 `-ct -s` 可直接为当前 HEAD 创建标签，跳过暂存、提交和推送流程
- 适用于仅需为现有提交打标签的场景，提高了灵活性
- 该模式下，若同时指定 `-cr`，也会直接创建对应的 GitHub Release

### 文档
- 更新了帮助文档，以明确区分完整工作流和独立模式
