# AutoTags v0.1.8（2025-10-29）

## 变更概览

- 新增版本排序、发布说明生成及变更日志链接节点

## 详情

### 新增
- `version_sort.rs`：解析并按版本号（vX.Y.Z）降序排列标签，获取最高版本
- `find_unpublished_tag.rs`：查找最新未发布（无对应 Release）的标签
- `find_previous_release.rs`：在指定标签之前定位最近的已有 Release 的标签
- `collect_changes.rs`：收集两个标签之间的所有标签信息及提交记录，兼容无前一次 Release 的场景
- `build_release_notes.rs`：调用 AI 分析节点生成 Release Notes，并在 AI 不可用时提供原始信息的降级方案
- `append_changelog_url.rs`：在 Release Notes 末尾追加 Full Changelog URL（基于 GitHub 模板）

### 重构
- 在 `autotag/src/nodes/compute/mod.rs` 中导出上述模块，对外提供统一入口
- 对工作流 `create_release.rs` 进行重构，将原来分散的代码抽象为上述节点，提升可读性、可维护性并保留原有的错误处理与降级逻辑
- 此改动不改动外部接口，仅内部实现细节变化，确保向后兼容
