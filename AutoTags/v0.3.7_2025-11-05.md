# AutoTags v0.3.7（2025-11-05）

## 变更概览

- 完善 Windows Shell 检测并优化输出信息

## 详情

### 改进
- 新增 detect_windows_shell 函数以可靠识别 PowerShell Core / Windows PowerShell / CMD
- 重构 print_shell_info：对 PowerShell 仅显示可执行路径；对 CMD 显示路径
- 修正显示逻辑，避免在非 CMD 环境输出 ComSpec 混淆
- 通过检测 PSModulePath、PSExecutionPolicyPreference、POWERSHELL_DISTRIBUTION_CHANNEL 等环境变量增强判断准确性

### 版本升级
- Cargo 版本从 0.3.7 升级至 0.3.8
