# AutoTags v0.0.29（2025-10-28）

## 变更概览

- 支持多种 release 创建模式
- 增强终端兼容性和 spinner 逻辑

## 详情

### 新增
- 将 `--create_release` 从布尔标志改为支持子选项的模式：
  - `-h/--highest`: 基于最高版本号创建 release（默认行为，向后兼容）
  - `-l/--latest`: 基于最新的未发布 tag 创建 release，并聚合变更日志
  - `-s/--set <tag>`: 为指定的 tag 创建 release
- 支持独立使用 `-cr` 命令，无需执行完整的提交流程即可创建 release

### 改进
- 增强终端兼容性
- 优化 spinner 完成逻辑
- 移除备选的打印逻辑

### 修复
- 修正 spinner 完成后的消息格式
- 移除进度条模板中冗余的消息占位符
