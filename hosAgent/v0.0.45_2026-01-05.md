# hosAgent v0.0.45 (2026-01-05)

## 变更概览

- 重构 AIConfigPanel 组件：将大文件拆分为模块化子组件结构
- 新增自定义 hooks：提取模型加载和配置操作逻辑
- 优化代码可维护性：提升组件复用性和代码组织结构
- 扩展 provider 支持：新增 NVIDIA 和 MiniMax 提供商

## 详情

### Added
- 新增 `frontend/src/app/console/AIConfig/components/` 目录：
  - `ApiKeySection.tsx`：API Key 配置区域
  - `ModelSelector.tsx`：AI 模型选择器
  - `AsrModelSection.tsx`：ASR 语音识别模型配置
  - `ParameterSection.tsx`：参数配置（温度/Token/流式/工具）
  - `ReasoningSection.tsx`：思考模式配置（从原文件迁移）
  - `SystemPromptSection.tsx`：系统提示词配置
  - `ConfigSummary.tsx`：当前配置摘要展示
  - `ConfigCodeBlock.tsx`：调用配置代码块（从原文件迁移）
- 新增 `frontend/src/app/console/AIConfig/hooks/` 目录：
  - `useModels.ts`：模型列表加载 hook
  - `useConfigActions.ts`：配置操作 hook（加载/保存/应用）
- 新增 NVIDIA 和 MiniMax provider 类型支持

### Changed
- 重构 `AIConfigPanel.tsx`：简化主组件，通过组合子组件实现功能
- 重构 `components/index.ts`：统一导出所有子组件
- 重构 `hooks/index.ts`：统一导出所有自定义 hooks
- 重构 `index.ts`：更新导出路径，适配新的目录结构

### Removed
- 移除 `frontend/src/app/console/AIConfig/ModelInfoCard.tsx`（已迁移至 components/）
- 移除 `frontend/src/app/console/AIConfig/ReasoningSection.tsx`（已迁移至 components/）
- 移除 `frontend/src/app/console/AIConfig/ConfigCodeBlock.tsx`（已迁移至 components/）

### Technical
- 组件拆分遵循单一职责原则：每个组件专注于特定功能区域
- Hook 抽取提升代码复用：模型加载和配置操作可在多处复用
- 保持向后兼容：外部引用路径通过 index.ts 统一管理
- 类型安全：完善 TypeScript 类型定义，确保组件间类型一致性
