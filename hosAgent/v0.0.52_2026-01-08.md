# hosAgent v0.0.52（2026-01-08）

## 变更概览

- 移除预设动画：删除所有预设动画资源和相关组件
- Sidebar 功能：添加全新的侧边栏组件，支持会话管理
- 前端重构：优化组件架构，简化 Header 和 Footer

## 详情

### Added
- 新增 Sidebar 组件，包含会话列表和会话项子组件
- 新增 sidebarStore 状态管理，管理 Sidebar 状态
- 新增 useSidebarSessions hook，处理会话相关逻辑
- 新增 messageMapper 工具函数，用于消息映射
- Footer 组件增强，优化上传菜单交互

### Changed
- 重构 MainContent 组件，集成 Sidebar 和聊天区域布局
- 优化 WelcomeScreen 组件，简化欢迎界面逻辑
- 优化 FloatActions 组件，调整浮动按钮行为
- 更新 chat store 初始状态

### Removed
- 删除预设动画资源（PresetAnimation 目录下所有 webm 文件）
- 移除 IP 相关组件（IPAnimation, PullRope）及相关资源
- 删除 Header 中的 ConfigStatus 和 TimeDisplay 子组件
- 删除 Header 相关的 hooks（useHeaderState）
- 移除不再使用的公共资源（next.svg, vercel.svg, pull-rope.png）

### Documentation
- 更新项目架构文档（.kiro/steering/architecture.md）
- 更新前端规范文档（.kiro/steering/frontend.md）

## 备注

- 本次重构大幅简化了前端组件结构
- 新增的 Sidebar 功能为后续会话管理提供基础
- 移除的动画资源为项目瘦身，减少不必要的资源占用
