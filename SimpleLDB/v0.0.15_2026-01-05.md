# SimpleLDB v0.0.15 (2026-01-05)

## 变更概览

- UI 整体优化：调整了主题样式、边距、字体和圆角，提升视觉一致性
- 导航增强：缩小了侧边栏宽度并新增了 API Docs 页面
- 数据管理优化：重构了数据管理布局、工具栏和对话框体验
- 仪表盘更新：新增了 API 信息展示组件

## 详情

### Added
- 新增 `web/src/app/docs` 页面及其布局组件，提供 API 文档入口
- 新增 `web/src/app/dashboard/ApiInfo.tsx` 组件，展示 API 访问信息
- 在 `Sidebar.tsx` 中新增 API Docs 导航项

### Changed
- 优化 `ThemeRegistry.tsx`：调整全局 `borderRadius` 为 8px，缩小默认间距，并优化了 `Typography` 字体大小
- 缩小 `Sidebar` 宽度从 260px 至 220px，并调整了内部元素的间距和字体
- 优化 `DataManagementLayout.tsx` 和 `DataTable.tsx`：采用更紧凑的布局设计
- 改进 `DataToolbar.tsx`：支持标签换行显示，优化按钮组布局
- 优化 `ImportExportDialog.tsx`：添加 `disableScrollLock` 以修复弹窗开启时的页面抖动

### Fixed
- 修复 `MuiPopover`、`MuiMenu`、`MuiModal` 等组件在 Windows 环境下的滚动锁定导致的布局偏移问题

### Removed
- 移除旧的 `web/src/app/data-management/DataStats.tsx` 组件

### UX
- 统一了全局组件的 `size="small"` 默认配置，使界面更显精致
- 优化了侧边栏项的悬停和选中状态视觉效果
