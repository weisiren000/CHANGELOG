# SimpleLDB v0.0.20（2026-01-29）

## 变更概览

前端架构大规模重构为模块化分层设计，同时优化后端 AI 服务的稳定性。

## 新增

### 前端架构重构
- **modules/** 目录：按业务模块组织组件
  - `ask/` - 问答模块（含搜索配置、AI 配置、响应展示）
  - `dashboard/` - 仪表盘模块
  - `data-management/` - 数据管理模块
  - `docs/` - API 文档模块
  - `settings/` - 设置模块
- **shared/** 目录：全局共享资源
  - `components/` - 布局组件、反馈组件、数据展示组件等
  - `hooks/` - 通用 Hooks
  - `store/` - 全局状态管理
  - `theme/` - MUI 主题配置
  - `types/` - 全局类型定义
  - `utils/` - 工具函数
- **config/** 目录：中心化配置
  - `routes.ts` - 路由配置
  - `menu.ts` - 侧边栏菜单配置

### 后端优化
- AI 服务（Groq、SiliconFlow）添加超时设置（60秒连接超时，10秒建立连接超时）
- 添加 max_retries=1 重试配置
- 增强错误处理和调试日志

## 变更

- 页面入口精简：各 `page.tsx` 从数百行缩减至约 5 行
- 删除旧目录：`src/components/`、`src/store/`、`src/lib/`、`src/types/`
- 所有导入路径更新为新的模块化路径

## 架构设计

依赖规则：
```
app/ → modules/ → shared/ → config/
```
