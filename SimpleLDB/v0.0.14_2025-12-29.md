# SimpleLDB v0.0.14（2025-12-29）

## 变更概览

- 前端主题与组件样式更贴近 MUI 默认风格：减少硬编码颜色/阴影，统一跟随 `theme.palette`、`divider` 与 `shadows`。
- 多个页面卡片与容器组件改为 `variant="outlined" + elevation={0}` 的轻量化视觉（头部、仪表盘、数据管理、设置页等）。
- 将 `CHANGELOG` 作为 git submodule 引入，便于版本日志独立维护。

## 详情

### Added
- `ThemeRegistry` 启用 `cssVariables`，并通过 `CssBaseline enableColorScheme` / `MuiCssBaseline.defaultProps.enableColorScheme` 让浏览器原生控件可跟随配色方案。
- 主题 `typography.fontFamily` 调整为系统字体栈，减少对特定字体安装的依赖。

### Changed
- `ThemeRegistry` 回归 MUI 默认主/辅色（`primary.main=#1976d2`、`secondary.main=#9c27b0`），背景默认色调整为更常见的 `#fafafa`，并移除自定义 `shadows`。
- `Header` 从彩色渐变头部改为半透明 `paper` 背景 + `backdropFilter`，hover/边框等样式统一使用 `action.hover` 与 `divider`。
- `Sidebar`、`layout` 等处从硬编码背景色改为使用 `background.paper` / `background.default`。
- `Ask` 搜索区域与设置按钮的背景/hover 统一改为基于 `theme.palette.*` 的 `alpha()` 计算。
- 仪表盘统计卡片（`StatsCards`、`RecordStats`、`RecentActivities`、`SystemInfo`）统一使用 outlined 视觉，并将图标/底色计算改为基于 `theme.palette`（包含更明确的颜色类型约束）。
- 数据管理页：表格分割线改用 `divider`，行 hover 与批量操作栏底色改为基于主色的 `alpha()`；搜索框聚焦描边阴影改为主题主色透明度计算。
- 设置页底部操作栏阴影改用 `theme.shadows[8]`，减少自定义 box-shadow 常量。

### Fixed
- 多处颜色/分割线/hover 从硬编码值切换为主题变量，降低不同主题/配色下的对比度与一致性问题。

### Removed
- 删除 `globals.css` 中的全局 body 背景色、自定义滚动条样式、全局过渡动画与 selection 配色（交由主题与组件样式统一管理）。
- 移除部分不必要的组件级 styleOverrides（例如 `TextField` 圆角、`Chip` 字重等）以贴近 MUI 默认行为。

## 备注
