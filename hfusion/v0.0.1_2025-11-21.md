# HFusion v0.0.1（2025-11-21）

## 变更概览

- 首次发布，实现完整的主题化系统
- 支持明暗模式一键切换
- 引入语义化 CSS 变量替代硬编码颜色

## 详情

### Added

- 新增 ThemeProvider 全局主题状态管理组件
- 新增 ModeToggle 主题切换按钮组件
- 新增语义化 CSS 变量系统（bg-background、text-foreground 等）
- 新增 View Transitions API 支持，实现平滑主题切换动画
- 新增 AI Handbook 开发规范文档
- 新增工作手册 WORK_MANUAL.txt

### Changed

- 将所有硬编码颜色值替换为语义化 CSS 变量

- 重构 Home 页面组件使用主题变量

### Documentation

- 新增主题化与样式规范章节
- 更新 AI Handbook AI 开发规范
