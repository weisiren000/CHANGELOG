# HFusion v0.0.12（2026-01-18）

## 变更概览

- 视觉升级：落地页使用真实图片替换渐变占位符，大幅提升视觉质量
- 交互增强：为 BentoGrid 卡片添加点击跳转功能，改善用户体验
- 路由优化：统一认证路由路径（/auth/login → /login）
- 数据完善：为工艺和工具数据添加封面图片字段

## 详情

### Added

- 为 BentoGrid 中的所有卡片添加 href 链接，支持点击跳转到对应页面
- 为 ShowcaseGrid 添加真实图片展示，支持 hover 缩放效果
- 为所有工艺分类（crafts）添加 coverImage 封面图片字段
- 为所有工具分类（tools）添加 coverImage 封面图片字段
- 新增 LabHero 组件，实现实验室页面的 Hero 展示区域

### Changed

- BentoGrid 组件：使用真实图片替换渐变占位符（来自 picsum.photos）
- ShowcaseGrid 组件：重构为纯图片展示，移除文字和图标元素
- AuthLayout 组件：使用高质量太空背景图替换原来的 globe.svg，添加深色遮罩层优化文字可读性
- LoginForm 组件：更新注册链接路径从 /auth/register 改为 /register
- MainNavbar 组件：更新登录和注册链接路径
- Hero 组件：添加 logoHref 参数避免循环引用问题
- CanvasChatPanel 组件：移除圆角样式（rounded-3xl → rounded-none）

### Removed

- GalleryCard 组件：移除未使用的 React 导入

## 备注

- 本版本主要聚焦于落地页视觉体验的全面提升
- 使用 picsum.photos 作为临时图片源，后续可替换为实际项目截图
- 路由路径统一简化，提升 URL 简洁性
