# visual-mcp v0.0.1（2025-01-19）

## 变更概览

- 项目初始化：完成 visual-mcp 项目的初始化和基础结构搭建
- 核心功能实现：实现了图片处理、视觉分析、MCP 服务器等核心功能
- 多模型支持：支持 OpenAI、SiliconFlow、Ollama、DeepSeek 等多种视觉模型提供商
- 部署支持：提供 Docker、Makefile 和多平台编译支持

## 详情

### Added

- **配置管理模块**
  - 实现从环境变量读取配置
  - 支持多种配置项：API Key、Base URL、Model、图片大小限制、传输方式等
  - 提供配置验证和默认值设置

- **图片处理模块**
  - 支持多种输入方式：本地文件、URL、Base64 编码
  - 实现智能图片压缩和缩放
  - 支持 PNG、JPEG、GIF、WebP 格式
  - 自动处理图片大小和尺寸限制

- **OpenAI 兼容 Provider**
  - 支持 GPT-4o、GPT-4-vision-preview 等模型
  - 支持 Claude 系列模型（通过 OpenAI 兼容接口）
  - 支持 DeepSeek-VL 模型
  - 支持自定义 Base URL 和 API Key

- **MCP 服务器**
  - 实现 MCP 协议标准服务器
  - 支持 stdio、HTTP、SSE 三种传输方式
  - 提供 vision 工具用于图片分析
  - 完善的错误处理和日志记录

- **项目文档和示例**
  - 完整的 README.md 文档
  - Dockerfile 和 docker-compose 支持
  - Makefile 构建脚本
  - 多平台交叉编译支持
  - 使用示例和配置说明

- **开发工具**
  - GitHub Actions CI/CD 配置
  - 代码测试和覆盖率报告
  - go fmt 和 go vet 集成
  - .gitignore 和 .gitattributes 配置

### Changed

- 更新 .gitignore，添加更完善的文件过滤规则
- 更新 AGENTS.md 和 CLAUDE.md 项目说明文档

### Removed

- 删除 goal.md 临时文档

## 备注

- 这是 visual-mcp 项目的首个版本，包含完整的 MCP 服务器实现和视觉分析功能
- 项目遵循 MCP 协议规范，可作为 Kiro、Claude Desktop 等 MCP 客户端的视觉能力扩展
- 推荐使用 SiliconFlow 的 GLM-4.6V 模型，性价比高且性能优秀
- 项目采用 Go 语言开发，支持 Windows、Linux、macOS 多平台部署
