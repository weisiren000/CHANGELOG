# v0.0.7 (2026-01-25)

## 变更概览

- 主要变更点 1：新增 look_pdf 工具，支持 PDF 文档视觉分析
- 主要变更点 2：支持多种 PDF 输入方式（本地文件、URL、Base64）
- 主要变更点 3：新增页面范围选择功能，优化大文档处理效率

## 详情

### Added
- 新增 look_pdf 工具，支持 PDF 文档视觉分析
- 支持本地文件路径、URL 和 Base64 编码三种输入方式
- 支持页面范围选择功能（如 "1-3,5,7-9" 格式）
- 集成 go-fitz 库进行 PDF 渲染为图片
- 新增 PDF 处理超时控制（5分钟）
- 新增 PDF 处理器，支持自动缩放和 Base64 编码输出

### Changed
- 更新 go.mod 和 go.sum，添加 PDF 处理相关依赖
- 更新 internal/tools/tools.go，注册 look_pdf 工具

### Technical
- 新增 internal/process/pdf.go，实现 PDF 处理核心逻辑
- 新增 internal/tools/look_pdf.go，实现 PDF 工具 MCP 接口
- 使用 golang.org/x/image/draw 实现高质量图片缩放
- 实现页面范围解析器，支持复杂页面选择语法

## 备注

- PDF 工具可以识别文档中的文字、图表、表格等信息
- 建议对于大文档使用 page_range 参数指定页面范围以提高处理效率
- 处理超时为 5 分钟，超时会提示用户优化处理策略
