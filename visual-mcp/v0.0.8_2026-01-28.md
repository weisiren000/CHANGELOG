# v0.0.8 (2026-01-28)

### Features

* **provider**: 优化错误重试逻辑
  * 改进 `isRetryableError` 函数，区分 4xx 客户端错误和 5xx 服务器错误
  * 4xx 错误（如 invalid api key、unauthorized、forbidden 等）不再重试
  * 5xx 服务器错误和网络错误（超时、连接错误等）继续重试

* **tools**: 添加资源使用限制
  * 图片数量限制：最多支持 20 张图片
  * PDF 数量限制：最多支持 5 个 PDF 文件
  * PDF 页数限制：最多支持 50 页

### Technical

* **process**: 提取 `ResizeImage` 函数优化代码结构
  * 将图片缩放逻辑提取为独立函数，提高代码复用性
